# Define Visual DNA (Visual System Lead)

This skill acts as the "Visual System Lead". Instead of relying on rigid template mapping, it uses **RAG (Retrieval-Augmented Generation)** technology to intelligently generate a unique visual system based on the Design Brief.

## Core Capabilities

1.  **Style Retrieval**: Retrieves the most matching visual language from a knowledge base containing 67+ professional UI styles (`styles.csv`).
2.  **Style Composition**: Capable of blending a "Primary Style" (e.g., Swiss Modernism) with multiple "Secondary Styles" (e.g., Glassmorphism Components).
3.  **Generative CSS**: Uses LLM to directly generate pixel-perfect `theme.css`, ensuring visual consistency and technical compliance.

## Usage

```bash
python3 my_skills/define_visual_dna/scripts/generate_dna.py <brief_path> <output_css_path>
```

### Parameters
*   `brief_path`: Path to JSON Design Brief generated by `create_design_brief`.
*   `output_css_path`: Path to save the generated `theme.css`.

## Resource Architecture

*   `references/styles.csv`: **Visual DNA Library**. Contains detailed CSS technical indicators for styles ranging from "Minimalism" to "Cyberpunk".
*   `prompts/visual_lead_system.md`: **Dynamic Generation Template** (English Active). Supports Jinja-style interpolation, used to build high-precision System Prompts.
*   `prompts/visual_lead_system.cn.md`: **Dynamic Generation Template** (Chinese Reference). Reference documentation.
*   `resources/master_template.html`: **[Deprecated] Atomic Master HTML**. Deprecated, HTML structure generation has been taken over by the `generate_assets` skill.
*   `scripts/generate_dna.py`: Core execution script (includes `StyleRetriever` and `StyleComposer` classes).

## Output Specifications

### 1. Theme CSS (`theme.css`)
Contains all dynamically calculated CSS variables:
*   **Color System**: 
    *   `--ppt-color-primary`, `--ppt-color-secondary`, `--ppt-color-accent`
    *   `--ppt-color-bg-main`, `--ppt-color-bg-surface`
    *   `--ppt-color-text-main`, `--ppt-color-text-secondary`
    *   `--ppt-bg-gradient-main`, `--ppt-gradient-accent`
*   **Typography**: `--ppt-font-family`, `--ppt-text-scale-base`, `--ppt-text-scale-ratio`
*   **Effects & Shapes**: 
    *   **Shadows**: `--ppt-shadow-card`, `--ppt-shadow-float`, `--ppt-shadow-glow`
    *   **Glass**: `--ppt-glass-bg`, `--ppt-glass-border`, `--ppt-backdrop-blur`
    *   **Borders**: `--ppt-border-light`, `--ppt-border-accent`
    *   **Spacing**: `--ppt-space-unit`
*   **Functional Classes** (A-F Category System):
    *   **A. Containers**: `.ppt-card-primary`, `.ppt-card-standard`, `.ppt-card-item`, `.ppt-card-overlay`, `.ppt-card-highlight`
    *   **B. Visual Decorators**: `.ppt-text-emphasis`, `.ppt-accent-marker`, `.ppt-divider`, `.ppt-connector`
    *   **C. Typography**: `.ppt-text-hero`, `.ppt-text-lead`, `.ppt-text-body-large/strong/small`, `.ppt-text-secondary`
    *   **D. Frame Decorators**: `.ppt-decorator-frame`, `.ppt-decorator-footer`
    *   **E. Shape Primitives**: `.ppt-shape-circle`, `.ppt-shape-rounded`, `.ppt-shape-pill`, `.ppt-shape-chevron`, `.ppt-shape-blob`
    *   **F. Background Utilities**: `.ppt-background-main`, `.ppt-background-contrast`, `.ppt-background-image`
